    Custom HTML Elements

    Input value refers to real DOM elements in this context.

    class MyElement extends HTMLElement {}
customElements.define("my-element", MyElement);

const v1 = document.createElement("my-element");
getType(v1) // "MyElement"
isCustomElement(v1); // true
isHtmlElement(v1); // true
isElement(v1); // true

// extending anything other than an HTML element (with "is")
class MyElement2 extends HTMLParagraphElement {}
customElements.define("my-element2", MyElement2, { extends: 'p' });

const v2 = document.createElement("my-element2");
getType(v2) // "HTMLElement"
isCustomElement(v1); // true
isHtmlElement(v1); // true
isElement(v1); // true
    Input value Return value Function
    <my-element> MyElement isCustomElement, isHtmlElement(), isElement()
    :--------------- :-------------------------- :----------------------------------------------------
    <my-element2> HTMLElement isCustomElement, isHtmlElement(), isElement()

    NodeList and HTMLCollection

    NodeList and HTMLCollection have a lot in common, but aren't exactly the same thing. If you need consistency between the two you can convert them to Arrays using Array.from().
